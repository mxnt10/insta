<h2><b>Shotcuts</b></h2>

- Ctrl+R: Reload webapp page.
- Ctrl+Q: Quit application.
- Alt+S: Open settings.
- Backspace: Back webapp page.

<h2><b>Installation Instructions</b></h2>

To install "Insta", use the following commands:
```sh
$ git clone https://github.com/mxnt10/insta.git
$ cd insta

$ sudo ./install.sh
```
